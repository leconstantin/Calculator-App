# Calculator-App
📌 Overview

The Calculator App is a simple and user-friendly application designed to perform basic mathematical operations. It helps users quickly solve calculations such as addition, subtraction, multiplication, and division.

🎯 Features
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
🧹 Clear button to reset calculations
📱 Responsive and easy-to-use interface
🛠️ Technologies Used
HTML (for structure)
CSS (for design and layout)
JavaScript (for functionality and logic)
🚀 How It Works
The user inputs numbers using the calculator buttons.
The user selects an operation (+, −, ×, ÷).
The app processes the calculation using JavaScript.
The result is displayed on the screen instantly.
